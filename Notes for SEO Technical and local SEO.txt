Technical SEO:
	Technical SEO focuses on optimizing a website’s infrastructure so search engines can crawl, understand, index, and rank pages efficiently.

	Robots.txt:
		Robots.txt is a simple text file placed in the root of a website that gives instructions to search engine crawlers about which pages to index or not index. It helps control crawler activity, manage server load, and prevent indexing of sensitive or unnecessary pages.

Error Code Shown In Search Engine:

301 - Webpage Moved Permanently

302 - Temporary Redirect

404 -  Page deleted or URL typo
	- A 404 Error occurs when the server cannot find the requested page or resource.
	  This means the URL exists in the browser, but the server has no matching page at that location.
	  Common reasons include deleted pages, broken internal or external links, and server misconfigurations. It basically means the URL exists, but there is no content at that location.
	
503 - Service Unavailable / Server overload or maintenance
	- A 503 error occurs when the server is temporarily unable to handle the request.
      Unlike other errors, 503 means the server is working, but it cannot process the request at that moment due to temporary overload or
      maintenance.

504 - Slow server response

Core Web Vitals: A set of metrics used by Google to measure website speed, stability, and user experience.

		Core Web Vital			 What It Measures					        Simple Explanation									Good Score
FCP (First Contentful Paint)	Initial loading speed		How fast the first visible content appears on the screen			under 1.8 sec
LCP (Largest Contentful Paint)	Loading performance			How fast the main content of the page loads                         under 2.5 sec
CLS (Cumulative Layout Shift)	Visual stability			How much the page layout moves unexpectedly while loading            under 0.1 sec
INP (Interaction to Next Paint)	Interactivity				How fast the page responds to user actions                          under 200 microseconds
